Nombre del proyecto Self Challenge Eslogan "Challenge Yourself. Build Better Habits." Concepto Self Challenge es una aplicación de gamificación enfocada en el desarrollo personal. Su propósito es ayudar a las personas a construir hábitos positivos mediante desafíos diarios, recompensas, experiencia (XP), niveles, logros y estadísticas de progreso. El usuario debe sentir que está jugando mientras mejora su vida. Objetivo principal Diseñar y desarrollar una aplicación web moderna que permita a los usuarios crear, completar y seguir desafíos personales mientras reciben recompensas visuales y motivacionales. Público objetivo Estudiantes. Universitarios. Jóvenes profesionales. Personas entre 16 y 40 años. Usuarios interesados en mejorar hábitos y productividad. Tecnologías permitidas Utilizar únicamente: HTML5 CSS3 JavaScript ES6 No utilizar: React Angular Vue Node.js Bootstrap jQuery Todo debe funcionar de forma estática para ser compatible con GitHub Pages. Estilo visual El diseño debe ser de nivel profesional, inspirado en aplicaciones como: Duolingo Nike Run Club Habitica Apple Fitness Notion Spotify Debe transmitir: Modernidad Motivación Energía Minimalismo Calidad premium Paleta de colores Color principal: #4CAF50 Color secundario: #2196F3 Color de éxito: #00C853 Color de advertencia: #FFC107 Fondo claro: #F8FAFC Texto principal: #1F2937 Tarjetas: blanco con sombras suaves y bordes redondeados. Tipografía Google Fonts: Poppins Experiencia de usuario La aplicación debe sentirse como una aplicación móvil moderna y no como una página web tradicional. Debe incluir: Animaciones suaves. Transiciones elegantes. Microinteracciones. Barras de progreso animadas. Tarjetas interactivas. Efectos hover. Diseño responsive. Excelente experiencia en móviles, tabletas y escritorio.
